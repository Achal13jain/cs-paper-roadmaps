# Security Policy

This is a static, single-file site with no backend, no user accounts, and no
data collection. The threat surface is small.

If you discover something concerning — for example, a prototype-pollution
vector in the rendering JavaScript, an XSS path through the paper data, or a
malicious link in an entry — please **don't open a public issue**. Instead:

- Email the maintainer at: `jainachal38@gmail.com`
- Or open a private security advisory:
  https://github.com/Achal13jain/cs-paper-roadmaps/security/advisories/new

Expected response time: within 7 days. There's no bug bounty — this is a
volunteer-run academic resource.

For broken paper links, factual corrections, or anything not security-related,
please use the regular [issue tracker](https://github.com/Achal13jain/cs-paper-roadmaps/issues/new/choose).
